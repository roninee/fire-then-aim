# 创建一个程序，只用python自带 tkinter。 功能：在windows桌面 的最前端显示自定义的文本，也就是提醒。要求如下：
# 1.只显示文本，去除标题栏
# 2. 文本的背景透明
# 3. 文本窗口右键点击，弹出可以“关闭窗口”的菜单
# 4. 文本的颜色默认是黄色
# 5. 因为没有标题栏，所以点击窗口可以拖拽移动窗口
# 6.默认显示时水平居中，垂直方向在顶部offset 20px
# 7.文本是多行的。
# 8. 文本是可以从本地文件reminder_top.txt读取的，注意分行显示，空行不显示。如果找不到文件或者内容是空，文本使用默认值“不要随便笑”
# 9.文本内容需要 像走马灯那样循环翻滚，但是方向是垂直翻滚，而不是水平。
# 10.注意：tkinter 并不支持 'transparent' 作为颜色名称 ，所以你要用wm_attributes('-transparentcolor', color)方法来实现，color应该是个特殊颜色，比如black
# 11.注意窗口初始化时候可能label还没“绘制”出来，所以返回的宽高是 0，导致窗口几乎不可见。所以解决方案：
# 用 root.update_idletasks() 后，应该再等一下（或更可靠地改为用 label.winfo_reqwidth() 和 label.winfo_reqheight() 取推荐大小），这样不会出现这个问题。
