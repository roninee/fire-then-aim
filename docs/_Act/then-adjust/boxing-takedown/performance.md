

## ① 恢复核心传导能力（最重要）

每天只需 8 分钟：

*核心“传导链”训练*（不是仰卧起坐！）：

1. 死虫 Dead Bug
     3组 × 每侧10次
     （恢复神经-核心连接）
2. Bird-dog
     3组 × 每侧10次
     （恢复背–臀传导）
3. Pallof Press（弹力带即可）
     3组 × 每侧12
     （反旋转核心 = 实战力量关键）
4. Farmer Carry 农夫行走
     30–45秒 × 3
     （核心 + 背 + 臀的整合训练）

连续 2–3 周，你会感觉“力量终于能传上来”。

## **② 让你的背部“活过来” = 改善摔跤发力、硬拉、拳击力量**

背薄 + 久坐 = 已关闭许多年。

解决方式不是硬练大重量，是：

*后链唤醒 3 件套（每次训练前做 5 分钟）：*

1. 髋关节铰链练习（**Hip Hinge**）
2. 臀桥 / **单腿臀桥**
3. 弹力带拉伸：**Face Pulls 或 Pull Aparts**

*这一点对你效果特别大*。

##  **恢复中年神经系统（协调、速度、爆发力）**

你需要少量但高质量的“神经训练”：

*每周 2 次，每次 8–10 分钟：*

- **小跳、跨步、小范围的轻跳**（不累但**神经活化**）
- 力量动作的 **3 次爆发**（如轻重量硬拉）
- 短距离**冲刺（20米**）**2–3 次**即可
- 轻松影子拳（专注速度，不专注力量）

40岁后这是提升运动表现最有用的训练。

**你的跑跳自信会在 2–4 周内回来。**

## **训练频率建议**

你现在最需要的*不是大强度，而是持续性*：

- **每周 3 次力量训练**（40 分钟就够）
- **每天 5–10 分钟核心与背部激活**
- **每周 2 次神经系统激活（短跳、速度）**
- **每天步行**

你会发现自己在 3–8 周内恢复到 30 岁的运动表现。
