# 日本和韩国都属于发达国家，但是相比于欧洲国家（比如法国），感觉普通人活得很累是因为什么？

不是西方国家，

是除了儒家文化圈的国民之外都不爱奋斗。

儒家文化圈包括中国、日本、韩国、新加坡、越南算半个吧。

都是往死里奋斗的类型，这种奋斗感是刻在骨子里的，真的就是为了奋斗而奋斗。

我做了好多年的汉语老师，教过的学生包括但不限于欧美、南美、非洲、东南亚各国。

都没有那么大的奋斗瘾。

儒家文化圈的奋斗，是有一种生存恐惧感的，不奋斗就是一种罪恶，不奋斗分分钟就会被社会淘汰，外加上儒家文化的尊卑有序，爬不上去就感觉会被别人踩在脚下。

很多时候，明明生活已经很好了，但是环境就会给你一种你TM不奋斗还是是个人吗？的感觉。

倒不是别的国家的人不会奋斗，很多看似不靠谱的国家的人奋斗起来也很拼命，但是确不像儒家文化圈的人这么焦虑，他们奋斗起来有点像是打游戏，为了打得好也能拼命，但是打不好也死不了，那就这样呗。

这事最诡异的是，是否选择当一个奋斗逼很多时候跟物质条件无关，甚至跟身处何地都无关。

中国人和韩国人都在美国把学区房发扬光大了，真的让人害怕。

想了半天，可能还是跟科举有关，好处是只要奋斗，那科举制真的可以帮一介草民出人头地，但是没有科举的国家，也就那样了，奋斗也没啥大用。

科举保证了人才流动，打破了阶级固化，但是代价是全民幸福感丧失，人人变成奋斗逼。

凡是皆有代价啊。

顺便说一句，听一些在美华人说，在美国的印度人现在学习学得最疯，这个到底是什么文化因素，想不通，还请大神解释。

# 日本和韩国都属于发达国家，但是相比于欧洲国家（比如法国），感觉普通人活得很累是因为什么？

从人均 GDP 来看，日本和英法德接近。但是从生活幸福感上，日本人累得跟狗一样，而英法德却活的很幸福。韩国成为发达国家后和日本差不多，都是很苦逼。还有香港、新加坡等东亚地区也是如此。

首先我这个感觉是否准确？

这个是由什么造成的？

若干年后中国步入发达国家，会走向哪个方向，而这又是什么决定的？

各人粗浅意见是，人均资源占有率，社会福利政策，民族文化几个方面，希望大神详解。

还得得从儒家文化上找吧。

日本、韩国、台湾、新加坡、香港，这些国家或者地区都很发达，但是生活在这些地方的人还是累得要死。

发现一个问题没有？这些地方都处于儒家文化圈。

从儒家文化衍生出的礼制制度，科举制度，道德治国制度，都造成了内卷。

儒家文化是不讲快乐的。

在儒家文化圈，快乐是原罪。

一切都是以所谓的正事为王道。

努力奋斗，勤能补拙是被社会认可和称赞的。

服从权威，不向权威提出质疑才能不被批判。

吃得苦中苦，方为人上人。

通过自己的努力，通过考试，成为人上人，回头来碾压曾经的仇人是儒家文化圈的普遍心态。

一切兴趣爱好都是不被鼓励的，除非这个爱好符合儒家的传统价值观。

10年前读《万历十五年》，有一个细节让我震惊。

万历小时候贪玩，张居正就天天数落他，让他学点正式，比如书法什么的。

后来万历沉迷书法，并有所小成，张居正看到了，就开始教育万历不要沉迷书法，并举出宋徽宗的例子。

皇帝都是如此，所以电子海洛因是有历史渊源的。

所以在儒家文化圈，打游戏是原罪。

其实围棋和DOTA其实没有本质区别，都是游戏。

DOTA受众更广，奖金更多，玩好DOTA在小伙伴的声誉更高。

但是围棋就是高雅游戏，DOTA就是电子海洛因。

  

跑题了，回来。

在儒家文化圈，没有爱好，没有多元价值观，尊崇权威，号召努力奋斗。

社会认可的成功道路其实就一条路——

死磕自己

  

死磕自己，那题海战术就成立了，没有童年，每天学习15个小时。

死磕自己，语言学习就简化成被单词了。

死磕自己，加班过劳死都是理所当然的。

  

为什么中日韩的奋斗逼如此之多，因为在儒家文化的语境下，只有奋斗一条路。

  

总之，儒家文化就是一个不让人快乐的文化，是一种从痛苦中寻找升华的文化。

同时的，也是一种升华后的不平等文化。

在儒家文化圈，所有人是天生有等级秩序的，通过死磕自己完成对其他人的超越，是每个人的梦想。  

这种洗脑如此之深，几乎让人忘了可以换一种玩法或者不玩的权力。


我想说，还是感谢互联网，感谢丰富的物质，让我们可以不那么努力。
