* 旋转发力，手腕甩动
* *先屈髋，再屈膝*
* 可以稍微侧身，库里就是侧身。——投篮后：手肘伸直，手腕下压
* 除了基本的细节外，其实我觉得下蹲*沉球*，之后举球到*出手点时再顺势用髋发力起跳*出手是最为简单直观的叙述了 之后就是依个人能力、习惯慢慢调整到能一条龙输出就好

* 前脚掌稍微靠前，不是完全并列
