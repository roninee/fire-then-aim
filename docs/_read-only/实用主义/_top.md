兴趣：读书，白日梦，空想，逻辑能力强，由于天赋好，我能接触的任何东西我都喜欢，甚至练过合气道，空手道，巴西柔术——大部项目我很快会超越最底层的普通人，但是容易满足和缺乏耐心限制了我的高度。
