#### 三、是否应该优先训练“叙事表达”还是“情感表达”？

但情感表达和叙事表达是相互作用的：

- 情感表达 让你的表达有温度，让别人感受到你的真实情绪。
- 叙事表达 让你的表达更有条理，让别人听得懂你在说什么。

- 叙事表达（并行训练）
  - 练习三句话讲完一件事。
  - **用“时间顺序”或“因果关系”复述故事。**
