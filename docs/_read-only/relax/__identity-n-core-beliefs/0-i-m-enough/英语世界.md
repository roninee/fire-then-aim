2. **模仿学习的内容选择**
   **David Goggins**：

   - **Commencement Speech（毕业演讲）**：这种演讲通常有明确的逻辑结构，适合练习演讲技巧，但内容可能偏正式。
   - **《Can’t Hurt Me》**（有声书）：Goggins的嗓音+故事很有吸引力，并且能让你内化他的思维方式。适合长时间听，边听边做其他事情。
