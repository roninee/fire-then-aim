## 五、判断你是否走在正确路上的 5 个“*体感指标*”

| 指标                                 | 正确体感               |
| ------------------------------------ | ---------------------- |
| ① 出拳时手臂几乎无力感               | 感觉像*手自己飞出去*   |
| ② 出拳落点*声音变脆*                 | *“啪”而不是“嘭”*       |
| ③ 出拳后*重心稳，身体不歪*           | 表示能量没被卡在身体里 |
| ④ 呼吸自然、*心率低*                 | 表示放松度高           |
| ⑤ 每拳之后身体会有*轻微“震”或“回弹”* | 表示力传导顺畅         |

如果你能同时满足三项以上，说明你走在正确路线上。

## 三、如何改进 & 利用这次酸痛

✅ **1. 出拳要“松出、紧收”**

- 出拳时放松，让身体*像鞭子一样甩出去*；
- 收拳时迅速但不“暴力”拉回；
- 你可以想象“拳像橡皮筋甩出再自动回弹”，而不是硬拽回来。

✅ **2. 注意核心稳定**

- 出拳时*收紧下腹*（尤其是肚脐以下那块区域），让*力量从地面传上来*；
- *不要靠下背去“拧*”，而是*靠髋部和腹斜肌的旋转*。
