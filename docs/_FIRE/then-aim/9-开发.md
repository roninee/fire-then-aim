## cloudflare的 R2  和 zero trust



### **你能否把敏感图片/视频存在 R2 然后发给朋友？**

✔️ **可以，但不推荐存敏感内容。需注意几点：**

### **① R2 的链接是真公网链接**

被微信监测到 URL 指向外国服务器，**可能被“安全提示”或阻断预览**。

### **② 内容是否违规仍由微信判断**

你只是把文件放在国外，并不能让微信“忽略违规内容”。

### **③ R2 外链在微信中可能被“拦截二维码”处理**

特别是带 `.mp4`、`.jpg` 这种直链文件。

------

### **更实用做法（建议）：**

✔️ **用 Cloudflare Workers 给 R2 做中转反代**
 让 URL 看不出源站，就不容易被微信拦截。

例子：

```
https://my-worker.example.workers.dev/xxxx
```

微信对这种域名的审核更宽松。

👉 大陆用户大量采用 **Workers + R2** 的原因就在这。
